# gocii

A dead-simple command line tool that lets you convert an image to ASCII art. Currently only works with PNG and JPEG formats because the Go `image` library only supports encoding/decoding for those lol.

## Install

```
go install github.com/tomasohCHOM/gocii@latest
```

## Develop

Clone the repository:

```
git clone https://github.com/tomasohCHOM/gocii
cd gocii
```

Run:

```
go run .
```
